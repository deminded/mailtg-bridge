## Безопасность
- NFR-SEC-01: файл сессии и файлы учётных данных имеют права доступа только для владельца (600).
  Индикатор — права файлов = 600; популяция — файл сессии и файлы секретов; точка съёма — проверка
  прав после инициализации.
- NFR-SEC-02: секрет-скан репозитория не находит секретов. Индикатор — число находок секрет-скана
  = 0; точка съёма — прогон секрет-скана репозитория.
- NFR-SEC-03: файлы сессии и секретов размещены вне дерева репозитория. Индикатор — число
  секретов/сессий под контролем версий = 0; точка съёма — обзор состава репозитория.

  зафиксирована (да/нет).
