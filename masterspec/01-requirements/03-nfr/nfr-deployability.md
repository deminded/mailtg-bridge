# НФТ: развёртываемость

## Производительность
- NFR-DEPLOY-01: развёртывание на чистой машине по инструкции завершается за ≤ 10 минут.
  Индикатор — время от получения репозитория до первого доставленного письма; популяция — чистая
  машина по инструкции поставки; окно — разовая установка; точка съёма — ручной замер по README.

## SLA / SLO
- Не применимо (разовая операция установки).

## Безопасность
- NFR-DEPLOY-03: поставка не требует внесения секретов в код или репозиторий при развёртывании
  (секреты — только в конфигурации вне репозитория). Индикатор — число секретов, вносимых в код при
  развёртывании = 0; точка съёма — обзор шагов README.

## Отказоустойчивость
- Не применимо на уровне развёртывания.

## Трассируемость и наблюдаемость
- NFR-DEPLOY-04: поставка включает пример конфигурации и инструкцию, достаточные для полного
  развёртывания без обращения к автору. Индикатор — доля обязательных параметров, покрытых примером
  конфигурации = 100%.

## Ограничения по данным
- NFR-DEPLOY-02: смена почтового провайдера или аккаунта выполняется правкой конфигурации без
  изменения кода. Индикатор — число изменённых файлов кода при смене провайдера/аккаунта = 0; точка
  съёма — сравнение дерева кода до/после смены.
- NFR-DEPLOY-05: поставка предоставляет минимум один готовый способ автономного запуска
  (контейнерная оркестрация ИЛИ таймеры операционной системы). Индикатор — число готовых способов
  запуска ≥ 1.
